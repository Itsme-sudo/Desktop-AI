# Local Python AI Assistant

A simple offline Python AI assistant that can:
- Open system applications (Calculator, Notepad, Command Prompt, VS Code, etc.)
- Open websites (Google, YouTube, GitHub)
- Respond to simple chat commands
- Run completely offline, no internet required
